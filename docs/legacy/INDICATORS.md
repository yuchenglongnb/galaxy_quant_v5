# 📊 GalaxyQuant 指标说明

## 竞价分析指标

### 基础指标

| 指标 | 公式 | 含义 | 使用场景 |
|------|------|------|----------|
| **竞价涨幅** | `(开盘价 - 昨收) / 昨收 × 100` | 集合竞价形成的涨跌幅 | 判断早盘情绪，开盘价=竞价结束价 |
| **收盘涨跌** | `(收盘价 - 昨收) / 昨收 × 100` | 全天涨跌幅 | 判断当日整体表现 |
| **实体涨跌** | `(收盘价 - 开盘价) / 昨收 × 100` | 日内涨跌幅 | 判断日内走势强弱 |

### 行业竞价指标

| 指标 | 公式 | 含义 | 使用场景 |
|------|------|------|----------|
| **竞价额** | `∑(行业个股开盘成交额)` | 行业竞价阶段资金流入 | 识别早盘资金关注方向 |
| **竞价占比** | `竞价额 / 全天成交额 × 100` | 竞价成交占比 | 判断早盘活跃度 |
| **反转幅度** | `收盘涨跌 - 竞价涨幅` | 日内反转程度 | 识别高开低走/低开高走 |

### 反转行业分类

| 类型 | 条件 | 含义 |
|------|------|------|
| 🔺**低开高走** | 竞价 < 0% 且 收盘 > 0.5% | 早盘看空，尾盘翻多 |
| 🔻**高开低走** | 竞价 > 0.5% 且 收盘 < 0% | 早盘看多，尾盘翻空 |

### ETF竞价状态

| 状态 | 条件 | 含义 |
|------|------|------|
| 🔥高开高走 | 竞价>0.5% 且 收盘>竞价 | 强势延续 |
| ⚠️高开低走 | 竞价>0.5% 且 收盘<竞价-0.5% | 冲高回落 |
| 🔺低开高走 | 竞价<-0.5% 且 收盘>竞价+0.5% | 超跌反弹 |
| ❄️低开低走 | 竞价<-0.5% 且 收盘<竞价 | 弱势延续 |
| ➡️平开震荡 | 其他情况 | 观望为主 |

### 相关性分析

**竞价-收盘相关性** = `corr(行业内个股竞价涨幅, 收盘涨幅)`

| 相关性 | 含义 | 策略建议 |
|--------|------|----------|
| > 0.6 | 强正相关，动量延续 | 竞价强的股票可追涨 |
| 0.3~0.6 | 中等相关 | 谨慎跟随 |
| < 0.3 | 弱相关，日内反转多 | 逆向策略可能有效 |

## 复盘分析指标

### 指数环境评分

| 条件 | 得分 |
|------|------|
| 涨幅 > 1.5% | +2 |
| 涨幅 0.5%~1.5% | +1 |
| 跌幅 0.5%~1.5% | -1 |
| 跌幅 > 1.5% | -2 |
| 放量(量比>1.5)上涨 | +0.5 |
| 放量(量比>1.5)下跌 | -0.5 |

**总分判断**:
- ≥3: 🟢 环境安全 (积极做多)
- ≤-3: 🔴 环境恶劣 (空仓观望)
- 其他: 🟡 震荡环境 (轻指数重个股)

### 个股趋势标签

| 标签 | 条件 | 含义 |
|------|------|------|
| 🚀 连板 | 今日涨停 + 昨日涨停 | 强势龙头 |
| 🚀 涨停 | 今日涨停 | 当日强势 |
| 💚 跌停 | 跌幅 > 9.5% | 当日弱势 |
| 🚀 放量新高 | 收盘>5日高点 且 量比>1.5 | 突破形态 |
| ⚠️ 顶部背离 | 收盘≈5日高点 且 量比<0.7 | 缩量新高，警惕 |
| ⚡ 反包昨日 | 昨跌今涨，收盘>昨开 | 反转信号 |
| 🔴 冲高回落 | 上影线>实体 | 抛压明显 |
| 🌊 二次探底 | 收盘≈5日低点 | 底部试探 |

### 行业/ETF趋势标签

| 标签 | 条件 | 含义 |
|------|------|------|
| 🔥 持续强势 | 连续两天涨>1.5% 且 有涨停 | 主线行业 |
| 🔥 连续领涨 | 连续两天涨>1.5% | 强势延续 |
| 🔥 爆发领涨 | 涨>3% 且 量比>1.2 | 当日爆发 |
| 🚀 放量突破 | 收盘>5日高点 且 量比>1.5 | 板块突破 |
| ⚠️ 量价背离 | 收盘≈5日高点 且 量比<0.7 | 警惕回调 |
| ⚡ 超跌反弹 | 昨跌>1% 且 今涨>0.5% | 反弹修复 |
| 🔴 高位承压 | 冲高回落 | 短期见顶 |
| 🌊 回踩支撑 | 收盘≈5日低点 | 支撑试探 |

### 行业强度

**强度公式** = `成交额(亿) × 涨幅(%)`

强度越高，说明板块既有资金流入，又有价格上涨，是当日主线候选。

## 核心龙头矩阵

### 筛选逻辑

**🚀龙头策略** (连板/涨停核心):
- 成交额 > 3亿
- 属于主线行业
- 今日涨停 或 昨日涨停
- 评分: 涨幅×10 + 涨停+100 + 连板+200

**🐘中军策略** (权重核心):
- 成交额 > 3亿
- 属于主线行业
- 涨幅 > 3%
- 评分: 成交额(亿) × 涨幅

## 使用建议

1. **早盘看竞价**: 关注竞价涨幅靠前的行业，判断当日热点方向
2. **盘中看反转**: 低开高走的行业可能有超跌反弹机会
3. **尾盘看相关性**: 高相关性行业可在次日竞价时跟随
4. **复盘看主线**: 强度高的行业是持续主线，龙头可持股
