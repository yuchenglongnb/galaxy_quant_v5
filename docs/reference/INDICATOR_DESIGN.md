# 指标设计

本文只描述“指标想表征什么市场状态”。技术实现、模块结构和接口细节见 [技术设计](TECHNICAL_DESIGN.md) 与 [数据接口指南](DATA_API_GUIDE.md)。

## 设计目标

GalaxyQuant 的指标围绕三个问题：

1. 早盘资金是否拥挤，是否存在高开兑现风险。
2. 低开方向是否有资金承接，是否存在反核或修复机会。
3. 盘后市场主线是否延续，哪些行业和个股值得继续跟踪。

系统不是自动交易器，指标输出用于辅助人工判断。

## 早盘竞价指标

| 指标 | 公式或口径 | 表征状态 | 使用场景 |
| --- | --- | --- | --- |
| 竞价涨幅 | `(开盘价 - 昨收) / 昨收 * 100` | 早盘集合竞价情绪 | 判断高开、平开、低开 |
| 实体涨幅 | `(收盘价 - 开盘价) / 昨收 * 100` | 日内真实承接或兑现 | 验证早盘信号是否成立 |
| 竞价额 | 行业个股开盘成交额汇总 | 早盘资金关注度 | 行业竞价排行 |
| 竞价额排名 | 按竞价额降序排名 | 资金拥挤程度 | CP/SA 权重 |
| OAR | 市场量比环境，当前以上证量比为基准 | 增量或缩量环境 | 缩量时放大诱多风险 |
| 5 日位置 | `(收盘 - 5日低点) / (5日高点 - 5日低点)` | 短线位置高低 | 高位防追高，低位看修复 |

## CP 指数

CP 即 `Crowding Pump`，用于表征“拥挤诱多风险”。

想捕捉的状态：

- 昨日或近期已经大涨。
- 今日竞价继续高开或不愿低开。
- 资金在高位集中，容易成为兑现窗口。
- 缩量环境下，高开更容易缺少后续承接。

触发条件：

| 条件 | 含义 |
| --- | --- |
| 竞价涨幅超过高开阈值 | 主板默认 0.3%，双创默认 0.5% |
| 昨日涨幅 > 2%，今日竞价不低开 | 昨日获利盘仍有兑现压力 |
| 昨日涨幅 > 3%，今日竞价高于 -0.3% | 大涨后没有充分分歧释放 |

核心公式：

```text
CP = (排名权重 * 5日位置 / OAR)
     * (1 + 昨日涨幅 * 10 * 昨日量比 / 1.5)
     * 100
```

信号解释：

| CP 区间 | 含义 | 操作含义 |
| --- | --- | --- |
| `< 60` | 诱多风险不足 | 不单独作为风险信号 |
| `>= 60` | 拥挤风险较高 | 谨慎追高，持仓看兑现压力 |
| `>= 80` | 高拥挤 | 更关注高开低走或冲高回落 |

## SA 指数

SA 即 `Support Absorption`，用于表征“低开承接/反核机会”。

想捕捉的状态：

- 昨日或近期有恐慌、阴线、调整。
- 今日竞价明显低开。
- 低开同时有资金成交，说明不是纯无量下杀。
- 若日内实体转强，低开可能变成洗盘或反核。

触发条件：

| 条件 | 含义 |
| --- | --- |
| 竞价涨幅 < -0.3% | 必须是真低开 |
| 竞价额 > 0 | 需要有资金参与 |
| 排名靠前 | 资金关注度更高 |
| 昨日实体阴线且放量 | 恐慌释放后更容易出现承接 |

行业 SA 公式：

```text
SA = 竞价额(亿)
     * 排名权重
     * (1 + abs(竞价跌幅) / 0.5)
     * 恐慌释放系数
```

ETF SA 公式：

```text
SA = 20
     * 低开系数
     * 超跌系数
     * 恐慌释放系数
```

信号解释：

| SA 区间 | 含义 | 操作含义 |
| --- | --- | --- |
| `< 50` | 承接强度不足 | 观察 |
| `>= 50` | 存在反核候选 | 关注 9:35 后是否站稳 |
| `>= 80` | 强承接或强分歧 | 需要结合趋势和分时确认 |

## 场景标签

| 场景 | 表征情况 | 典型解释 |
| --- | --- | --- |
| `TRAP_HOT_SECTOR` | 热门板块高开拥挤 | 昨日追涨资金可能兑现 |
| `TRAP_MOMENTUM` | 连涨后高位高开 | 动能衰竭风险 |
| `TRAP_VOLUME_FADE` | 放量后缩量高开 | 后续资金不足 |
| `REVERSAL_PANIC` | 放量阴线后低开承接 | 恐慌出清后的反核 |
| `REVERSAL_WASHOUT` | 趋势中低开拉升 | 洗盘后延续 |
| `REVERSAL_OVERSOLD` | 低位低开承接 | 超跌反弹 |
| `TREND_CONTINUE` | 竞价后实体继续走强 | 趋势延续 |
| `TREND_ACCELERATE` | 放量实体大幅走强 | 趋势加速 |

## 复盘指标

| 指标 | 表征状态 | 使用场景 |
| --- | --- | --- |
| 指数环境评分 | 市场整体风险偏好 | 判断仓位和进攻性 |
| ETF 资金风向 | 主题方向强弱 | 识别资金偏好 |
| 行业强度 | `成交额(亿) * 涨幅(%)` | 找当日主线 |
| 龙头矩阵 | 主线行业内强势股票池 | 次日跟踪候选 |
| 连板/涨停标签 | 情绪强度 | 判断短线风格 |
| 放量新高 | 趋势突破 | 主升或加速候选 |
| 缩量新高 | 顶部背离风险 | 警惕回落 |
| 反包昨日 | 趋势修复 | 观察持续性 |

## 信号验证口径

当前复盘报告采用简化验证：

| 信号 | 成功标准 |
| --- | --- |
| 诱多 | 当日实体涨幅 `< 0` |
| 反核 | 当日实体涨幅 `> 1%` |

这个验证用于评估信号方向是否有效，不等同于实际交易收益。
